1. React js is a library of javascript
2. React js is used to create a dynamic application
3. React js is used to create a single page application (SPA)
4. React js is used one way data binding
5. React js create seo friendly website
6. React js create a responsive UI
7. React js used some modules of packages or library to create any web application
8. React js is provides re-useables components
9. React js used jsx (javascript XML) means create html inside of react js in format of .jsx
10. React js used props and state to access data one components to another components.
11. React js create any dynamic or server side web apps frontend.

## How to install react js app?
1. download node js in your systems
2. check node js version `node -v`
3. check NPM and NPX version `npm -v` or `npx -v`
4. create an react js app using `create-react-app` command

     **How to create app using create-react-app**
     1. check `node js`
     2. check `npm` or `npx` version
     3. `npx create-react-app appname`

     **Note: appname should be always in small case**

     4. after create app how can we run app
     5. `cd appname`
     6. npm start
     7. `http://www.localhost:3000`

     **How to create app using vite package**
     1. check `node js`
     2. check `npm` or `npx` version
     3. `npm create vite@latest`
     4. create project name appname
     5. select `framework`
     6. select `language javascript`
     7. after that click no
     8. after that click yes
     9. create a server and run app
     10. `http://localhost:5173/`
     11. cd appname
     12. `npm run dev`


     ****

     **Note: used create-react-app method is slow**
     **Note: used create vite@latest method is fast**

     **vite is a simple package of react js that can be fast create react js app**
