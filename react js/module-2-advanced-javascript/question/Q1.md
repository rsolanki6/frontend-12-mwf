# What is JavaScript?

**Javascript** is a high-level, interpretend programming language used to make web pages interactive and dynamic.

## what Javscript Does
Javascript runs in the browser (and also on server) and lets you:
- Respond to user actions (clicks,typing,scrolling)
- Update web content without reloading the page
- Create animations and interactive UI elements
- Communicate with servers using APIs

## Where Javasctipt Runs
- **Web browsers** (chrome,Firefox,safari,etc.)
- **Servers** using environments like **Node.js**
- **Mobile apps**, **desktop apps**, and even **IoT devices**

## Key features
- **Dynamic typing** - variable types are determined at runtime
- **Event-driven** - reacts to user and system events
- **Cross-plateform** - works on all modern browser
- **First-class functions** - functions can be stored, passed and returned

## Javascript vs HTML & CSS

| Techonology | Purpose |
|-------------|---------|
| HTML  | Structure of the webpage |
| CSS | Styling and layout |
| Javascript | Behavior and logic |
