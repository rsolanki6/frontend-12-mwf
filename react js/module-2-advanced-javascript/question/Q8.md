# What is NaN?

**NaN** stands for **"Not a Number."**

It is commonly used in programming (Javascript, Python, etc.) to represent:
- An invalid numeric result
- An undefined mathemetical operation (e.g., `0/0`)
- A failed numeric conversion (e.g., converting `"abc"` to number)

## Example in Javascript

```javascript
0/0 // Nan
Number("hi") // NaN
```
