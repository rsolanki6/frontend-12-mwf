# what is JSON server in react js ?

1. JSON server is a simple light weight tool that allows you to create a fake REST API using a JSON file as the data source.

2. It is commonly used in React developement for testing and prototyping purpose, as it provides a quick and easy way to simulate API endpoint without needing to set up a backend server.


**How to install JSON server**

1. 
