# What is CLI command In React Js? 

**CLI** stands for **Command Line Interface**.
in React JS, CLI commands are **terminal or command prompt commands used to create, run, and manage React applications**.

Instead of using a graphical interface, developers type commands in the terminal to perform tasks.

## Common react CLI Commands

### 1. Create a React Applications
This command creates a new React project.

```bash
npx create-react-app my-app
```

- `npx` runs the package without installing it globally. 
- `create-react-app` is a tool used to set up a React project.
- `my-app` is the name of the project.



### 2. Move into the Project Folder
```bash
cd my-app
```
This command opens the project directory.


### 3. Start the React Development Server
```bash
npm start
```
This runs the React application and opens it in the browser, usually at:
```
https://localhost:3000
```


### 4. Install Packages
```bash
npm install axios
```
This command installs additional libraries in the React project.


### 5. Build the Application
```bash
npm run build
```
This creates an optimized production version of the React application.

### 6. Run Tests
```bash
npm test
```
This command runs tests for the React project.
