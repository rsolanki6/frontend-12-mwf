# How to install NPM and How to check version of NPM?

## what is NPM?

**NPM (Node Package Manager)** is a package manager for **Node.js**.
It is used to **Install, manage, and update libraries or packages** used in javascript and React applications.

NPM is automatically installed when you install **Node.js**.

---

# 1. How to Install NPM 

## Step 1: Install Node.js

1. Go to the offical Node.js website `https://nodejs.org`
2. Download the **LTS (Long Term Support)** Version.
3. Run the installer and follow the installation steps.
4. Once Node.js is installed, **NPM is installed sutomatically**.

---

# 2. How to Check NPM Version
After installing Node.js, open **Command Prompt (Windows)** or **Terminal (Linux/mac)** and run the follwing command:

```bash
npm -v
```
OR
```bash
npm --version
```
If NPM is installed correctly, it will display the **Version number**.

Example Output:
```
9.6.7
```

---
# 3. Check Node.js Version (Optional)
You can also check the node.js version using:
```bash
node -v
```

Example output:
```
v18.16.0
```
